Pętle w JavaScript

Pętla for jest używana, gdy wiemy, ile razy chcemy wykonać kod. Składa się z trzech części:

for(zainicjowanie_zmiennej; warunek_kończący_wykonywanie_pętli; zmiana_zmiennej) {
  kod który zostanie wykonany pewną ilość razy
}

Pętla while wykonuje kod tak długo, jak długo warunek jest spełniony:

while (wyrażenie_sprawdzające_zakończenie_pętli) {
  ...fragment kodu który będzie powtarzany...
}

Pętla do...while wykonuje kod co najmniej raz, a następnie powtarza go tak długo, jak długo warunek jest spełniony:

do {
  ...fragment kodu który będzie powtarzany...
} while (false)

Informacje o Pętlach

Pętle w programowaniu są kluczowym narzędziem do wykonywania powtarzalnych zadań. Pozwalają na wielokrotne wykonywanie tego samego fragmentu kodu, co może znacznie uprościć i skrócić kod.

for - Najlepiej stosować, gdy liczba iteracji jest znana.

while - Używamy, gdy nie znamy liczby iteracji, ale znamy warunek, który musi być spełniony.

do...while - Podobna do while, ale kod wykona się przynajmniej raz, bez względu na spełnienie warunku.
